# HBD To Ming 🎂

โปรเจกต์เว็บไซต์อวยพรวันเกิดสุดพิเศษ ที่ออกแบบด้วยความตั้งใจ พร้อมแอนิเมชัน 3D การ์ด และระบบสุ่มข้อความอวยพร

## 📂 โครงสร้างโปรเจกต์ (Project Structure)

โปรเจกต์นี้ถูกจัดระเบียบตามมาตรฐานเพื่อให้ดูแลรักษาและปรับปรุงต่อได้ง่าย

- `index.html` - ไฟล์หลักของเว็บไซต์ (โครงสร้างเนื้อหา)
- `style.css` - ไฟล์สำหรับจัดการดีไซน์ สีสัน และแอนิเมชันทั้งหมด
- `script.js` - ไฟล์สำหรับจัดการระบบ Interactive (การคลิกเปลี่ยนหน้า, พลุกระดาษ, พิมพ์ดีด)
- `assets/` - โฟลเดอร์สำหรับเก็บทรัพยากรต่างๆ
  - `images/` - เก็บรูปภาพ เช่น `salisa.png`
  - `videos/` - เก็บวิดีโอพื้นหลัง เช่น `aurora.mp4`

## 🚀 วิธีการใช้งาน

1. โหลดไฟล์ทั้งหมดเก็บไว้ในโฟลเดอร์เดียวกัน
2. ดับเบิลคลิกเปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ (แนะนำ Chrome หรือ Safari)
3. หากต้องการเปลี่ยนรูปภาพ ให้นำรูปใหม่มาตั้งชื่อว่า `salisa.png` แล้วนำไปทับของเดิมในโฟลเดอร์ `assets/images/`
4. หากต้องการเปลี่ยนข้อความอวยพรในการ์ด สามารถเปิดไฟล์ `index.html` ด้วยโปรแกรม Notepad หรือ VS Code แล้วค้นหาจุดแก้ไขที่ระบุไว้
